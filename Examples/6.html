OpenLayer 3 Designer - Example map: Clusters and Statistics

This example map has been automaticly created by the OpenLayers 3 Designer. No code was written mannualy to produce this page, only button pushing was involved.

This map has a couple of clustering examples.  Popups are also enabled on this layer, when a feature is selected that contains many clustered points, the user can pick which one to view

This map shows the California Airports with a simple cluster style. The labels are 'Counts' of the total number of clustered points. Labels can be Count, Min, Max, Mean or a Sum of the cluster field. 

Labels will never appear in the key.

The cluster tolerance is greater in this map than the one to the right. higher cluster tolerances mean features are more likley to cluster.

This map clusters all the hotels in florida and styles the points based on the total possible occupancy of all the hotel points in the cluster. The points are clustered using the "Cluster and statistics" sytle with the 'Sum' statistic type.  Therefore the points are styled on the total number of hotels rooms in a given cluster, this "Sum" increases as the cluster gains more points when zooming out.

The Cluster tolerance is set to just over half of that in the map to the left.
