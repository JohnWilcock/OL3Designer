OpenLayer 3 Designer - Example map: Dynamic Styles

This example map has been automaticly created by the OpenLayers 3 Designer. No code was written mannualy to produce this page, only button pushing was involved.

This map contians only one layer, but has many pre-selected styles.  The user picks which style they wish to apply via the drop down menu in the key.  The styles are applied dynamicly to the map based on the attribute data of the features.

The data for this map was obtained from Caltrans GIS.
